# ILL WILL — Audio Source & License Ledger

All shipped SFX derive from **CC0 / public-domain** sources only. CC0 requires no
attribution; this ledger is kept for provenance and auditability, not obligation.
No asset here carries a share-alike, attribution, or non-commercial restriction.

Processing (resample to 44.1 kHz 16-bit, silence trim, 2–5 ms raised-cosine
edge-fades to bit-exact zero, per-family peak normalization) is done by
`tools/sfx_process.ps1`. Processing does not change the license — CC0 output stays CC0.
