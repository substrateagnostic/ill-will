# Meshy.ai custom assets — license note

The `.glb` models in this directory were generated with **[Meshy.ai](https://www.meshy.ai)**
text-to-3D, under Alex Galle-From's **paid** Meshy account.

## Usage rights

Per Meshy's terms for paid subscribers, assets generated on a paid plan are owned
by the account holder and may be used for commercial and non-commercial projects,
including within this game ("Par for the Curse" / the UN Party Game anthology).

- Generator: Meshy.ai text-to-3D (model: `meshy-5`, low-poly mode; the
  `generated/` subfolder batch is `meshy-6`, low-poly mode)
- Account: paid subscription (Alex Galle-From)
- Scope: usable in this project (distribution as part of the compiled game / repo)

## Provenance

Each asset's exact generation prompt and Meshy task IDs are recorded in
`docs/verify/meshy-assets-VERIFY.md` (props batch),
`docs/verify/cosmetics-VERIFY.md` (`cosmetics/` hat batch),
`docs/verify/visual-polish-VERIFY.md` (polish batch: crate, armchair,
crusher_head, spinner_arms, pendulum_blade, market_stall, broken_column,
pedestal_fan) and `docs/verify/meshy-forge-VERIFY.md` + `tools/meshy_forge_report.json`
(`generated/` batch: 32 graves/awards/board/estate props, model `meshy-6`,
generated by `tools/meshy_forge.ps1`; extended 2026-07-16 with 10 NPC-troupe/
board-dressing props, same tool + model, `docs/verify/meshy-troupe-VERIFY.md`).
Assets were generated preview → refine, exported as GLB, and imported into
Godot 4.6. `executor_butler_idle.glb` is the shipped `executor_butler.glb`
run through Meshy's auto-rigging + preset-idle animation endpoints
(`tools/meshy_rig_trial.ps1`, task ids in `tools/meshy_rig_trial_report.json`).
